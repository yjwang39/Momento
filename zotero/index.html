How to sync PDFs across devices?

  1. Choose a cloud service provider that allows you to sync files across devices.
  2. Change settings in Zotero to acticate relatice paths for linked attachments.
    Go to Preferences -> Advanced -> Files and Folders,
    1. Change Data Directory Location from ‘Default’ to a custom Directory folder under your synced directory. For example, I create a zotero folder in Nutstore, then I change to /zotero/Directory.
    2. Set Linked Attachment Base Directory to a folder in another Attachment folder under the synced directory, e.g. /zotero/Attachment.
      *Note: ‘You cannot select a Linked Attachment Base Directory within the Zotero data directory.’ *
  3. Change settings in ZotFile.
    Go to Tools -> ZotFile Preferences,
    1. Change Source Folder for Attaching New Files to /zotero/Directory/storage.
    2. Change Location of Files to /zotero/Attachment.
    3. (Optional but recommended) Activate subfolder defined by /%T/%y/%j (item type/year/journal)
  4. Move your attachments to Attachment folder (also sorted).
    Ctrl + A to select all the items and right-click, then choose Manage Attachments -> Rename Attachments.
